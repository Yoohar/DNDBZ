# DNDBZ Character Menu — Web Edition

Character manager for the DNDBZ homebrew system (DBZ × D&D 5.0). This repo hosts the playable web build (Godot 4.7.1 HTML5 export) via GitHub Pages.

## For playtesters

Open the page, wait for the loader, accept the fan-content notice, and build a character: pick a species, heritage, and class, assign ability scores, then hit **Finalize Build** to see the playable sheet (HP, combat statistics, AC, techniques, forms).

**Known limitations while development continues:**

- **Pick counts and one-time choices aren't enforced yet.** The pickers show you what you're eligible for (with prerequisites and lock reasons), but nothing stops you over-picking — e.g. the "pick ONE General or Species God Form when you first receive God Ki" rule is stated in the picker text, not enforced. The rules text shown in each picker is authoritative; treat the app like a smart paper sheet.
- **Locked options can be taken deliberately.** Clicking a locked (prerequisite-unmet) option once shows a DM-permission warning; clicking again takes it anyway. That's a feature (DM overrides), not a bug — but it means the app trusts you.
- **Some content is still being wired up**: a handful of forms and soulbind-granted abilities aren't fully connected yet, and feat/technique-embedded sub-choices (e.g. picking your Android Part upgrade) may not surface a picker.
- **Saves live in your browser** (per-browser storage). Clearing site data clears your characters.

Found something broken or confusing? Tell the dev — screenshots help.
